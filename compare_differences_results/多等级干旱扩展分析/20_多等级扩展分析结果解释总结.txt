
多等级干旱扩展分析结果解释总结
===============================================================================

一、分析目的
-------------------------------------------------------------------------------
本模块是在原Extreme vs Normal主分析基础上的稳健性与扩展分析。核心问题是：
不同SPI干旱等级相对Normal时，森林15 cm微气候温度对ERA5 2 m温度的响应斜率
是否升高，即森林温度缓冲能力是否减弱；同时检查这种变化是否随干旱等级增强
呈现严格剂量响应。

二、主扩展LMM结果
-------------------------------------------------------------------------------
四个干旱等级分别相对Normal的Delta_CBI结果：
- 轻度干旱：Delta_CBI = 0.0250 (95% CI: 0.0193 to 0.0307; FDR p < 0.001; n_site_months = 159, n_sites = 27)
- 中度干旱：Delta_CBI = 0.0108 (95% CI: 0.0036 to 0.0180; FDR p = 0.00425; n_site_months = 114, n_sites = 26)
- 重度干旱：Delta_CBI = 0.0130 (95% CI: 0.0011 to 0.0249; FDR p = 0.0322; n_site_months = 44, n_sites = 23)
- 极端干旱：Delta_CBI = 0.0333 (95% CI: 0.0135 to 0.0531; FDR p = 0.00195; n_site_months = 16, n_sites = 13)
概括：4/4个等级的Delta_CBI为正，4/4个等级在FDR 0.05水平下显著。

主扩展模型的Delta_CBI排序为：
中度干旱(0.0108) < 重度干旱(0.0130) < 轻度干旱(0.0250) < 极端干旱(0.0333)

解释：Delta_CBI > 0表示目标干旱等级下CBI高于Normal，即微气候温度更强地随
宏气候温度变化，森林温度缓冲能力相对减弱。若四个等级均为正且显著，可表述为：
不同干旱等级相对Normal均表现出森林温度缓冲能力减弱。

三、MacroSD敏感性分析
-------------------------------------------------------------------------------
额外要求Target与Normal状态内ERA5 SD达到门槛后的结果：
- 轻度干旱：Delta_CBI = 0.0250 (95% CI: 0.0193 to 0.0307; FDR p < 0.001; n_site_months = 159, n_sites = 27)
- 中度干旱：Delta_CBI = 0.0108 (95% CI: 0.0036 to 0.0180; FDR p = 0.00425; n_site_months = 114, n_sites = 26)
- 重度干旱：Delta_CBI = 0.0130 (95% CI: 0.0011 to 0.0249; FDR p = 0.0322; n_site_months = 44, n_sites = 23)
- 极端干旱：Delta_CBI = 0.0333 (95% CI: 0.0135 to 0.0531; FDR p = 0.00195; n_site_months = 16, n_sites = 13)
概括：4/4个等级的Delta_CBI为正，4/4个等级在FDR 0.05水平下显著。

解释：该敏感性分析用于检查主扩展结论是否依赖状态内ERA5温度变异不足的Site_Month。
若方向、量级和显著性与主扩展结果一致，可表述为：
主要发现对状态内宏气候温度变异筛选具有稳健性。

四、有序趋势与剂量响应判断
-------------------------------------------------------------------------------
- Main_hours_only: 加权趋势斜率 = -0.0035 (95% CI: -0.0276 to 0.0206; p = 0.597); Spearman rho = nan; 严格递增 = 否。
- Main_hours_only: 加权趋势斜率 = nan (95% CI: nan to nan; 未计算); Spearman rho = 0.4; 严格递增 = 否。
- Sensitivity_hours_plus_MacroSD: 加权趋势斜率 = -0.0035 (95% CI: -0.0276 to 0.0206; p = 0.597); Spearman rho = nan; 严格递增 = 否。
- Sensitivity_hours_plus_MacroSD: 加权趋势斜率 = nan (95% CI: nan to nan; 未计算); Spearman rho = 0.4; 严格递增 = 否。

解释：有序趋势检验回答的是更强的问题，即Delta_CBI是否随Mild、Moderate、Severe、
Extreme的顺序整体升高。即使各等级相对Normal均显著，只要Delta_CBI点估计不严格递增，
就不应写成“干旱越严重，缓冲越弱”的严格剂量响应。

五、非单调性诊断
-------------------------------------------------------------------------------
月份组成诊断：
- 轻度干旱样本最多的月份为12月，占该等级小时数的17.1%。
- 中度干旱样本最多的月份为7月，占该等级小时数的22.7%。
- 重度干旱样本最多的月份为8月，占该等级小时数的39.0%。
- 极端干旱样本最多的月份为4月，占该等级小时数的46.0%。

站点贡献诊断：
- 极端干旱：Top1站点贡献=13.0%，Top3站点贡献=36.0%，有效站点数=10.73。
- 轻度干旱：Top1站点贡献=6.2%，Top3站点贡献=16.8%，有效站点数=24.92。
- 中度干旱：Top1站点贡献=7.0%，Top3站点贡献=20.6%，有效站点数=22.27。
- 重度干旱：Top1站点贡献=7.4%，Top3站点贡献=21.2%，有效站点数=20.01。

共同支持域诊断：
- All_selected_classes：共同月份数=3，共同站点数=10，建议可做限制样本分析=True。
- Mild_vs_Extreme：共同月份数=3，共同站点数=13，建议可做限制样本分析=True。
- Mild_vs_Moderate：共同月份数=8，共同站点数=26，建议可做限制样本分析=True。
- Mild_vs_Severe：共同月份数=6，共同站点数=23，建议可做限制样本分析=True。

解释：这些诊断用于判断多等级排序不严格单调时，是否可能受到月份分布、站点贡献
或共同支持域差异影响。它们是解释性诊断，不是新的显著性检验。

六、限制样本稳健性分析
-------------------------------------------------------------------------------
- All_class_common_month_site | All_selected_classes: 轻度干旱=0.0588; 中度干旱=0.0184; 重度干旱=0.0401; 极端干旱=0.0591
- Pairwise_common_month_site | Mild_vs_Extreme: 轻度干旱=0.0461; 极端干旱=0.0333
- Pairwise_common_month_site | Mild_vs_Moderate: 轻度干旱=0.0350; 中度干旱=0.0108
- Pairwise_common_month_site | Mild_vs_Severe: 轻度干旱=0.0306; 重度干旱=0.0130

解释：限制样本LMM只限制共同月份和共同站点，仍然拟合“目标等级 vs Normal”。
其目的不是替代主扩展模型，而是检查在更可比的支持域内，Delta_CBI方向和非单调模式
是否仍然存在。

七、可用于论文的结果表述
-------------------------------------------------------------------------------
在控制站点、月份背景和宏气候温度条件后，不同干旱等级相对Normal总体表现为
正的Delta_CBI，说明干旱条件下森林微气候温度对宏气候温度变化的响应增强，
即森林温度缓冲能力减弱。多等级扩展分析表明，这一结论并不局限于极端干旱，
而是在多个干旱等级中均可观察到。

然而，多等级Delta_CBI的点估计排序不应自动解释为严格剂量响应。只有当Mild到Extreme
呈现一致递增、置信区间和趋势检验均支持时，才适合使用“随干旱强度增强而逐步减弱”
这类强表述。若本次结果不满足严格递增，应采用更谨慎的表述：
“各干旱等级均表现出缓冲减弱，但不支持严格单调的干旱强度梯度响应。”

八、解释边界
-------------------------------------------------------------------------------
1. 主结论应以05号主扩展LMM结果为核心，08号MacroSD结果和16号限制样本结果作为稳健性证据。
2. 09号趋势检验是结果层面的辅助判断，不能替代小时级LMM。
3. 11-15号非单调性诊断用于解释样本组成与可比性，不应被写成新的因果检验。
4. 16-19号限制样本分析仍是“目标等级 vs Normal”，不能直接表述为某个干旱等级显著强于另一个等级。
